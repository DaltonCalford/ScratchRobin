# Decision Record 0001: Stage-Gated Delivery

## Date

2026-02-18

## Decision

Delivery is stage-gated, with explicit exit criteria per stage.

## Rationale

1. Reduces migration risk.
2. Makes scope and readiness measurable.

## Consequences

1. Features cannot bypass earlier contract gates.
2. Release readiness is tied to conformance evidence, not feature count.
