# Delivery Stages

## Stage 1: Foundation

1. Project skeleton and CI baseline
2. Backend contracts for parser/adapter/server-session boundaries
3. Core docs and conformance matrix initialization

## Stage 2: UI Parity Shell

1. Main window and panel structure parity
2. Object browser and editor frame scaffolding
3. Command/action routing to adapter APIs

## Stage 3: Backend Integration

1. Real native adapter protocol bridge
2. Session lifecycle, authentication, and capability discovery
3. Metadata and query workflow integration

## Stage 4: Feature Conformance

1. Feature matrix completion against ScratchBird capabilities
2. Regression suite hardening
3. Performance baseline and release readiness
