# Data Reference

**Status**: Draft  
**Last Updated**: 2026-02-05

---

## Purpose

Defines the data reference panel used in reporting to browse tables, fields, and descriptions.

---

## Features

- Field list by schema/table
- Field types and descriptions
- Sample values (limited)
- Usage hints (joins, common filters)
- Links to models and metrics

---

## Rules

- Data reference is read-only.
- Field descriptions can be edited only by users with Design permission.
- Changes are stored as project metadata.
