# NoSQL Language Specifications (Beta)

**[Back to NoSQL Beta Requirements](../README.md)**

This directory contains full language specifications (BNF/EBNF + usage guidance)
for NoSQL dialects that ScratchBird may emulate. Each language spec is intended
to be standalone and complete.

## Index

### Document
- MongoDB Query Language (MQL): `mongodb_mql/SPECIFICATION.md`
- CouchDB Mango Query: `couchdb_mango/SPECIFICATION.md`
- Couchbase N1QL (SQL++): `couchbase_n1ql/SPECIFICATION.md`
- ArangoDB AQL: `arangodb_aql/SPECIFICATION.md`

### Key-Value
- Redis Commands (RESP): `redis_resp/SPECIFICATION.md`

### Wide-Column
- Cassandra CQL: `cassandra_cql/SPECIFICATION.md`
- HBase Shell + Filter Language: `hbase_shell/SPECIFICATION.md`

### Graph
- Cypher / openCypher: `cypher/SPECIFICATION.md`
- Gremlin (TinkerPop): `gremlin/SPECIFICATION.md`
- SPARQL: `sparql/SPECIFICATION.md`

### Search
- Elasticsearch / OpenSearch Query DSL: `elasticsearch_dsl/SPECIFICATION.md`
- Lucene Query Syntax: `lucene_query_syntax/SPECIFICATION.md`

### Time-Series
- InfluxQL: `influxql/SPECIFICATION.md`
- Flux: `flux/SPECIFICATION.md`
- PromQL: `promql/SPECIFICATION.md`

### Vector
- Milvus Query Language (SQL-like): `milvus_query/SPECIFICATION.md`
- Weaviate GraphQL: `weaviate_graphql/SPECIFICATION.md`
