# Application Support

**[← Back to Beta Requirements](../README.md)** | **[← Back to Specifications Index](../../README.md)**

This directory contains specifications for popular application and platform integrations.

## Overview

ScratchBird provides compatibility with widely-used applications and platforms, enabling drop-in database replacement for content management systems, e-commerce platforms, GIS systems, and more.

## Application Specifications

### Content Management Systems

- **[wordpress/](wordpress/)** - WordPress (43% of websites)
- **[drupal/](drupal/)** - Drupal
- **[joomla/](joomla/)** - Joomla

### E-Commerce

- **[magento/](magento/)** - Magento
- **[woocommerce/](woocommerce/)** - WooCommerce (WordPress plugin)

### Geographic Information Systems

- **[qgis/](qgis/)** - QGIS (Open-source GIS)
- **[geoserver/](geoserver/)** - GeoServer (Geospatial server)

### Collaboration

- **[mattermost/](mattermost/)** - Mattermost (Team collaboration)

### Business Applications

- **[odoo-erp/](odoo-erp/)** - Odoo ERP
- **[metabase/](metabase/)** - Metabase (Business intelligence)
