This directory contains specifications for future features planned but not yet scheduled for Alpha or Beta releases.

## Overview

These specifications document features that may be implemented in future releases after Beta, including advanced APIs, error handling frameworks, and experimental features.

## Key Features (Future)

### C API

Native C API for embedding ScratchBird:

- **Embedded database** - Embed ScratchBird in applications
- **Zero-copy operations** - Efficient data access
- **Thread-safe API** - Concurrent access support
- **Memory management** - Explicit memory control

### Error Handling

Advanced error handling framework:

- **Error codes** - Structured error code system
- **Error context** - Rich error context and stack traces
- **Error recovery** - Automatic error recovery mechanisms
- **Debugging support** - Enhanced debugging information

## Status

These features are **not currently scheduled** for implementation. They represent potential future directions for ScratchBird development.
