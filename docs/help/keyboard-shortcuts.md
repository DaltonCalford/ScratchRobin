## Custom Shortcuts

You can customize shortcuts in **Preferences → Keyboard Shortcuts**.

## Platform Differences

### macOS
Replace `Ctrl` with `Cmd` (⌘) key:
- `Cmd+N` instead of `Ctrl+N`
- `Cmd+S` instead of `Ctrl+S`

### Linux/Windows
Use `Ctrl` key as shown above.
