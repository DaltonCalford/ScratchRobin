# SQL Editor Shortcuts

Keyboard shortcuts are planned for a future iteration. The current build uses
menu actions and toolbar buttons for execution and export.

Planned defaults (subject to change):
- Run statement: Ctrl+Enter
- Cancel statement: Esc
- Format SQL: Ctrl+Shift+F
- Find: Ctrl+F
