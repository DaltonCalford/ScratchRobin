# ERD Notation Dictionaries

**Status**: Draft  
**Last Updated**: 2026-02-05

---

## Purpose

Defines the canonical symbols, labels, and meanings for ERD notations supported by ScratchRobin.

---

## Supported Notations

- Crow's Foot
- IDEF1X
- UML Class-style
- Chen
- Silverston

---

## Example (Crow's Foot)

| Symbol | Meaning |
| --- | --- |
| `|` | One |
| `o` | Optional |
| `<` | Many |

---

## Edge Cases

- If a diagram is converted between notations, cardinality must be preserved.
