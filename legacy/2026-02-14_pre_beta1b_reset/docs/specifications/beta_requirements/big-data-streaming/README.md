# Big Data & Streaming Integrations

**[← Back to Beta Requirements](../README.md)** | **[← Back to Specifications Index](../../README.md)**

This directory contains specifications for big data processing and streaming platform integrations.

## Overview

ScratchBird integrates with major big data and streaming platforms for large-scale data processing, real-time analytics, and ETL workflows.

## Specifications

### Distributed Computing

**[apache-spark/](apache-spark/)** - Apache Spark
- Spark SQL connector
- DataFrame API support
- Batch and streaming processing
- **Market Share:** Dominant big data processing

**[apache-flink/](apache-flink/)** - Apache Flink
- JDBC connector
- Stream processing
- Exactly-once semantics
- **Market Share:** Leading stream processor

### Event Streaming

**[apache-kafka/](apache-kafka/)** - Apache Kafka
- Kafka Connect integration
- Change Data Capture (CDC)
- Sink and source connectors
- **Market Share:** Dominant event streaming
 - Spec: `apache-kafka/SPECIFICATION.md`

### Hadoop Ecosystem

**[hadoop-ecosystem/](hadoop-ecosystem/)** - Hadoop Integration
- Hive compatibility
- HBase integration
- Pig Latin support
- **Market Share:** Legacy big data standard

### ETL Platforms

**[etl-platforms/](etl-platforms/)** - ETL Tools
- Talend integration
- Pentaho Data Integration
- Informatica PowerCenter
- **Market Share:** Enterprise ETL standard
