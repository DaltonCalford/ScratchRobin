## Base Prompt Template

```
Create a clean, modern SVG icon for a database administration application.

SPECIFICATIONS:
- Style: Flat design with subtle depth, minimal shadows
- Size: 24x24 viewBox
- Stroke width: 1.5px
- Corner radius: 2px for small elements, 4px for large
- Colors: Use the specified color palette
- Background: Transparent
- Format: SVG with clean, optimized paths

COLOR PALETTE:
- Primary: #2E86AB (Database Blue)
- Secondary: #FF6B35 (Robin Orange - for highlights)
- Neutral: #495057 (Dark Gray)
- Background elements: #E9ECEF (Light Gray)

DESIGN PRINCIPLES:
- Use simple geometric shapes
- Clear silhouette at small sizes
- Consistent visual weight
- No text or letters in icons
- Modern, professional appearance

ICON DESCRIPTION: [Describe the icon here]
```
